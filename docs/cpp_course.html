Kursa Programma

1. C++ Pamati un Atmiņas Pārvaldība

  • Pointeri un references
  • Stack vs heap atmiņa
  • RAII un smart pointeri
  • STL konteineri un algoritmi

2. Objektorientēta Programmēšana C++

  • Klases un inheritance
  • Polimorfisms un virtual funkcijas
  • Move semantika
  • Design patterns (Factory, Observer)

3. Daudzpavedienu Programmēšana

  • Thread un mutex
  • Race conditions un deadlocks
  • Async programmēšana
  • Lock-free algoritmi

4. Spēļu Izstrāde ar Unreal Engine

  • C++ un Blueprint integrācija
  • Game loop un fizika
  • AI pamati
  • Optimizācijas paņēmieni

Kas Ietverta Kursā?

  • 3 pilnvērtīgi projekti portfolio (spēle, multithread aplikācija, sistēmu programmatūra)
  • Mentoru sesijas ar nozares ekspertiem
  • Git versiju kontroles sistēma no pirmās nodarbības
  • Karjeras diena ar vadošajiem Latvijas tech uzņēmumiem
  • Mācību materiāli un kodu piemēri pēc kursa
